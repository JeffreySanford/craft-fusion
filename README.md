# Craft Fusion

Craft Fusion is an Nx monorepo with Angular 19 (Material Design 3 Expressive), NestJS, and Go.

## Quick start

```bash
pnpm install

pnpm dlx nx serve craft-web
pnpm dlx nx serve craft-nest
pnpm dlx nx serve craft-go
```
